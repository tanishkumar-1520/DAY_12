****Brief Description :****

Notes Managemnet System is a modern Full-Stack web application designed to help individuals log and track their daily activities, thoughts, and essential tasks. Built with a robust Node.js/Express backend and a dynamic, responsive vanilla JavaScript frontend, it serves as a lightweight personal micro-journaling platform. 


The app features a stunning visual layout with modern premium elements like translucent glassmorphic cards and floating dynamic gradients. Users can write fresh entries, view sorted records dynamically, search/filter logs by specific dates effortlessly, and perform instant CRUD operations without any full-page reloads. 



##✨ Key Features :



#- Full CRUD Functionality:


Seamlessly create, read, update, and delete daily entries with instant feedback.  


#- Real-time Date Filtering:


Quickly find past entries using the built-in, real-time date filtration system. 


#- Advanced File-Based Storage:


Uses local JSON storage structure managed robustly via backend file streams to safely log user entries.  


#- Premium Glassmorphic UI:


Built with a beautiful dark mode layout featuring dynamic background blur effects and fluid desktop/mobile grid systems.  


#-. Clean API Architecture:


Follows RESTful principles using Express.js middleware and full CORS support for rapid and secure communication.  


##🛠️ Tech Stack Used :


#- Frontend:**


HTML5, CSS3 (Custom Variables, CSS Grids, Flexbox), Vanilla JavaScript (ES6, Async/Await APIs).  


#- Backend:**


Node.js, Express.js. 


#- Dependencies:**


cors (Cross-Origin Resource Sharing), fs (Native File System for JSON logging).
